# 工具脚本

此目录包含各种辅助工具脚本。

## 脚本列表

- `quick_start.py` - 快速测试 SIGGRAPH 信息获取（无需 API Key）
- `redownload_paper.py` - 重新下载论文
- `cleanup_obsolete_dirs.py` - 清理废弃的目录结构
- `setup_skill_structure.py` - 设置技能目录结构

## 使用方法

```bash
# 快速开始（测试 SIGGRAPH）
python src/scripts/quick_start.py

# 重新下载论文
python src/scripts/redownload_paper.py

# 清理废弃目录
python src/scripts/cleanup_obsolete_dirs.py

# 设置技能结构
python src/scripts/setup_skill_structure.py
```
